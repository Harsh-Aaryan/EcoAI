Build EcoCore — a Smart City residential energy + AI compute PWA.

AESTHETIC: Calm, hopeful, biophilic. The feeling of a sustainable 
future that is already arriving. Inspired by forest light, morning 
clarity, and Scandinavian environmental design. No neon. No glow. 
Soft luminosity and organic warmth throughout.

COLORS (CSS variables):
--bg:         #0d1a12   (deep forest night)
--surface:    #122218   (dark moss)
--border:     #1e3a28   (shadowed fern)
--green:      #4ade80   (spring green, primary)
--green-soft: #86efac   (sage, secondary)
--sun:        #fbbf24   (morning sunlight, warm accent)
--sky:        #67e8f9   (clear water, cool accent)
--text:       #e2f5e9   (pale leaf)
--muted:      #5a7d65   (lichen)

TYPOGRAPHY:
- Fraunces (Google Fonts) — display/headers, weight 300–700
  Organic, humanist serif. Feels grown not manufactured.
- DM Mono — all numbers, data, prices, carbon values, timestamps

TEXTURE:
- SVG feTurbulence grain filter applied at 4% opacity over all cards
  Creates recycled paper / uncoated stock warmth
- No scanlines, no glitch, no hard edges

ANIMATIONS:
- Breathing pulse on live indicators: scale 1→1.04→1, 3s ease infinite
- Water-rise fill on progress bars: width animates from 0 on mount
- Leaf drift: single decorative SVG leaf floats slowly top→bottom 
  on the home screen background, 20s linear infinite, 5% opacity
- Page transitions: gentle fade + 6px translateY, 300ms ease
- All animations feel slow, deliberate, calm

TECH STACK:
- React 18 PWA (Vite + vite-plugin-pwa)
- Auth0 React SDK — universal login, JWT, user profile in MongoDB
- MongoDB Atlas — collections: users, jobs, tasks, grid_snapshots, earnings
- FastAPI (Python) — REST endpoints, auth middleware
- Celery + Redis — AI job queue and scheduling
- Leaflet.js + react-leaflet + OpenStreetMap tiles
  Tile layer: CartoDB Positron (light, calm, earthy)
  Home nodes: CircleMarker in spring green (#4ade80), 
  breathing scale animation via CSS, popup shows battery/earnings/carbon
  Map container: soft border, no harsh styling, 220px height
- WattTime API — marginal carbon intensity by grid region
- ERCOT / EIA API — live spot prices + 24h forecast
- Open-Meteo API — solar irradiance forecast
- Ollama + Groq — AI inference runtime
- Recharts — sparklines and bar charts, no sharp colors, 
  use --green and --sun for fills
- Tailwind CSS utility classes
- Web Push API — calm notification copy ("Your home earned $2.10 
  during last night's clean window ☀️")
